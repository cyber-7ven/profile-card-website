# 👥 Profile Card Website

##  Project Description
This is a group project created by a team of 5 members. The goal of this project is to build a simple and clean profile card website using Tailwind CSS.  

The main page displays multiple profile cards, and each card links to a dedicated profile page that contains full details about the individual.

---

##  Features
- Multiple profile cards on the homepage  
- Each card includes image, name, and role  
- Clickable cards that link to individual profile pages  
- Dedicated profile pages with detailed information  
- “Back to Home” navigation on each profile page  
- Responsive design using Tailwind CSS  
- Clean UI with hover effects  

---

##  Tech Stack
- HTML5  
- Tailwind CSS

##  How It Works
- The homepage displays all team members as profile cards  
- When a user clicks on a card, it opens that member’s profile page  
- Each profile page shows more details about the person  
- Users can navigate back to the homepage easily
